Discuss Vamp plugins and the plugin SDK

There is a Github project for the Vamp Plugin SDK. For problems with the Vamp SDK, consider opening an issue on the Github issue tracker.

Sadly the dedicated online discussion forum for Vamp plugins has now closed, after we gave up fighting spam registrations. The forum content is still online, but as a static mirror of the original forum: it's no longer possible to register, log in, or post. You can find the static mirror here.
